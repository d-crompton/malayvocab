Noun Prefixes

It is possible to turn a base word into a noun (kata nama) by adding a prefix. Each prefix is used to create a different type of noun.

PE-

Nouns that start with PE- are names for "a person that does something"

Lari (To Run) Pelari (Runner)
Nyanyi (To Sing) Penyanyi (Singer)

KE-

Nouns that start with KE- are names for "a person who embodies the meaning of the base word" or are the subject of the base word.

Kasih (To Love) Kekasih (Lover)
Tua (Old) Petua (Elder/Chief)

JURU-

Nouns that start with JURU- are names for "a person directly involved with the work associated with the base word". However, this does not apply to all jobs.

Terbang (To Fly) Juruterbang (Pilot)
Wang (Money) Juruwang (Cashier)

MAHA-

The prefix MAHA- is used to describe "something grand and of the highest level". It originates in Sanskrit and Hindu influence but because of Islam in Malaysia, maha- is almost exclusively applied to God.

Guru (Teacher) Mahaguru (Grand Teacher)
Raja (King) Maharaja (Supreme Ruler)
